
# method 1 - use cargo cmd

build debug version
```
cargo build
```

run
```
cargo run
```

build release version
```
cargo build --release
```

build and run
```
cargo run --release
```

# method 2 - use make cmd

```
make build      編譯 release 版本
make run        執行 release 版本
make rebuild    乾淨重建
make clean      清除 build 資料
```
